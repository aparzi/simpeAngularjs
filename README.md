# simpeAngularjs

Simple app in angularjs

<br />
<b>PRE REQUIREMENTS:</b>
<li>Install npm</li>
<li>Install nodejs</li>
<li>Install bower</li>
<br />
<b>SETUP:</b>
<li>clone this project</li>
<li>navigate to the project folder</li>
<li>bower install angular#1.6.3</li>
<li>npm start</li>
<b>Enjoy</b>
