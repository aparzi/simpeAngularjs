# simpleAngularjs

Simple app in angularjs

<br />
<b>PRE REQUIREMENTS:</b>
<li>Install npm</li>
<li>Install nodejs</li>
<li>Install bower</li>
<br />
<b>SETUP:</b>
<li>clone this repository</li>
<li>npm install</li>
<li>navigate into src folder and type the command: bower install angular-material</li>
<li>npm start</li>
<b>Enjoy</b>
